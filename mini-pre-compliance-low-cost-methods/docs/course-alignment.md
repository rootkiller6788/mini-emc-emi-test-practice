# Nine-School Curriculum Mapping

| School | Course | Topics Covered |
|--------|--------|---------------|
| **MIT** | 6.630 EM Waves | Near-field/far-field wave impedance, antenna fundamentals |
| **Stanford** | EE359 Wireless | Antenna factor, Friis equation, measurement uncertainty |
| **Berkeley** | EE117 EM Fields | Maxwell's equations for near-field probes, mutual inductance |
| **Illinois** | ECE 451 EM | TEM cells, shielding effectiveness, plane wave theory |
| **Michigan** | EECS 411 Microwave | Cable effects, impedance matching, VSWR measurements |
| **Georgia Tech** | ECE 6350 EM | Site attenuation, NSA theory, OATS validation |
| **TU Munich** | High-Frequency Eng. | Spectrum analyzer theory, detector models, measurement automation |
| **ETH** | 227-0455 EM | CISPR standards, uncertainty budgets, compliance methodology |
| **Tsinghua** | 电磁场与电磁波 | Near-field probes, LISN design, EMC fundamentals |
