# Course Prerequisite Dependency Tree

```
mini-pre-compliance-low-cost-methods
├── mini-emc-basics (LISN, TEM cell, antenna factor concepts)
├── mini-fourier-analysis (spectrum analyzer IF processing)
├── mini-circuit-analysis (impedance networks, voltage dividers)
├── mini-communication-principle (dB/dBm/dBuV conversions)
├── mini-electromagnetic-wave (near-field, wave impedance, Friis)
├── mini-sensor-measurement (probe theory, calibration methods)
└── mini-signal-system-theory (noise floor, MDS, system sensitivity)
```

## Knowledge Dependencies
1. **dB/dBm/dBuV conversions** (mini-communication-principle) — required for all amplitude calculations
2. **Fourier analysis** (mini-fourier-analysis) — required for spectrum analyzer understanding
3. **Faraday's / Biot-Savart laws** (mini-electromagnetic-wave) — required for near-field probes
4. **Antenna theory** (mini-electromagnetic-wave) — required for antenna factor and radiated emissions
5. **Statistics** (mini-digital-signal-process) — required for uncertainty and correlation analysis
6. **Transmission line theory** (mini-electromagnetic-wave) — required for TEM cell and impedance matching
